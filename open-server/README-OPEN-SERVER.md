# ASU Open Server Deployment Kit

Version 0.2.0 preview

## Environment

- Open Server Panel 6.5.1
- Apache
- PHP >= 8.3
- MySQL 8.4

## Project path

```
C:\OSPanel\home\asu.local
```

## Installation

Run:

```powershell
START-ASU-INSTALL.ps1
```

Flow:

1. Diagnostics
2. Fresh install / Upgrade
3. Test
4. Report generation

## Deployment layout

```
payload/
installer/
reports/
```
