ASU Open Server Panel 6.5.1

Требования:
- Open Server Panel 6.5.1
- Apache
- PHP-8.5
- MySQL-8.4

Установка:

1. Создать каталог:
C:\OSPanel\home\asu.local

2. Скопировать содержимое каталога payload в созданный каталог.

3. Проверить наличие файла:
C:\OSPanel\home\asu.local\.osp\project.ini

4. Перезапустить Open Server Panel.

5. Проверить появление проекта:
asu.local

6. Открыть в браузере:
http://asu.local
