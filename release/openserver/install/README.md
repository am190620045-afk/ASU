# ASU Open Server Installation

Complete installation sequence:

1. Read 01-install.md
2. Configure database using 02-database.md
3. Perform first launch using 03-first-run.md
4. Use 04-troubleshooting.md if errors occur

Target environment:

- Open Server Panel 6.5.1
- Apache
- PHP 8.5
- MySQL 8.4

Compatibility:

- Minimum PHP: 8.3
- Recommended runtime: PHP 8.5
