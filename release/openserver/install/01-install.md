# Open Server Installation

## Environment

- Open Server Panel 6.5.1
- Apache
- PHP 8.5
- MySQL 8.4

## Steps

1. Copy ASU into Open Server domains directory.
2. Create local project domain: asu.local.
3. Set web root to ASU/public.
4. Copy environment template to .env.
5. Install Composer dependencies.
6. Create database.
7. Run runtime checks.

Do not expose internal directories through Apache.
