# Step 03 - First Run

Checklist:

1. Start Apache and MySQL in Open Server Panel.
2. Open ASU local domain.
3. Run environment check:

```
/openserver/check-environment.php
```

4. Verify:

- PHP 8.5 runtime
- Database connection
- Application boot
- Storage permissions

Target:

ASU running locally through Open Server Panel 6.5.1.
