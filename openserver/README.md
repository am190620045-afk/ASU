# ASU Open Server Preview

Target environment:

- Open Server Panel 6.5.1
- Apache
- PHP 8.5
- MySQL 8.4

## Compatibility

Minimum supported PHP version: 8.3+

Recommended runtime:

- PHP 8.5
- MySQL 8.4
- Apache

## Installation

1. Copy ASU into Open Server domains directory.
2. Configure Apache document root to the `public` directory.
3. Create database.
4. Copy environment configuration.
5. Run Composer dependencies installation.
6. Open the local domain.
